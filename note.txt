git
 - 로컬 저장소 형상 관리 (gitbash : git 명령어 터미널)
 - commit : 저장하다
 - branch : 분기하다, 가지를뻗다, 갈래, 줄기 (master > main > others... test)

github
 - 원격 저장소 형상 관리. 코드에디터도 있다.
 - push : 밀어 넣다
 - pull : 땡겨 오다 (충돌시 자동으로 merge를 시도하고 사용자에게 해결책을 제시한다.)
 - fetch : 갖다 붙이다. 가져오다 (충돌시 자동으로 merge 하지 않는다.)
